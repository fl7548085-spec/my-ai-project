# 시각 인식 기반 수도꼭지 물 사용 상태 인식 프로젝트

## 프로젝트 소개
본 프로젝트는 인공지능 기반의 시각 인식 기술을 활용하여 수도꼭지의 물 사용 상태를 판별하는 웹 프로젝트이다.  
주요 인식 대상은 **잠금**, **출수**, **낙수**의 세 가지 상태이며, 실제 사용 환경에서 나타나는 시각적 차이를 바탕으로 각 상태를 구분한다.

본 프로젝트는 수도꼭지의 물 흐름 상태를 이미지 분류 방식으로 인식하고, 이를 통해 절수 알림 및 스마트 제어 시스템으로 확장 가능한 기초 연구 프로토타입을 제안하는 것을 목적으로 한다.

## 프로젝트 목표
본 프로젝트의 목표는 수도꼭지의 물 사용 상태를 시각적으로 식별하고, 해당 인식 방식이 실제 생활 환경에서 적용 가능한지 검증하는 것이다.  
이를 바탕으로 향후 절수 시스템, 이상 상태 감지, 자동 제어 기술과의 연계 가능성을 탐색하고자 한다.

## 인식 대상
본 프로젝트에서는 다음의 세 가지 상태를 주요 분류 대상으로 설정하였다.

- **잠금**: 수도꼭지가 닫혀 있으며 물의 흐름이 없는 상태
- **출수**: 수도꼭지에서 물이 정상적으로 지속적으로 나오는 상태
- **낙수**: 수도꼭지가 완전히 잠기지 않아 물방울 또는 미세한 물줄기가 계속되는 상태

## 프로젝트 활용 목적
본 프로젝트는 다음과 같은 용도로 활용될 수 있다.

- 수도꼭지 물 사용 상태의 시각 인식 실험
- 절수 시스템을 위한 초기 프로토타입 검증
- 이미지 분류 모델 기반의 웹 시연 및 테스트
- AI 시각 인식 관련 수업 과제 및 발표 자료
- 향후 IoT 기반 자동 알림 및 제어 시스템의 기초 모듈

## 프로젝트 특징
- 시각 인식 기반으로 구현되어 별도의 복잡한 센서 설치가 필요하지 않음
- 잠금, 출수, 낙수의 세 가지 상태를 중심으로 명확하게 분류
- 경량화 모델을 적용하여 비교적 간단한 테스트와 배포가 가능
- GitHub Pages를 통해 온라인 시연 및 공유 가능
- 과제 제출, 발표, 프로토타입 검증용 프로젝트로 활용하기 적합함
